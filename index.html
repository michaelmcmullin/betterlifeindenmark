How OECD countries compare to each other

The World Economic Forum tweeted a graph showing that Denmark has the best work-life balance out of all the OECD countries. The original data came from the OECD Better Life Initiative, 2016.

This data has been reproduced here as a bar chart. Each value was scaled to a range between 1 and 100, representing the lowest and highest figure in each category respectively. From there, the OECD average was calculated, and each bar represents how far above or below the average a particular country scores in each category. Red bars indicate bad results: sometimes this means below the OECD average (e.g. income), sometimes above average (e.g. homocides).
